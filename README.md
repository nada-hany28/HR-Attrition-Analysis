# HR Employee Attrition Analysis

## Problem
Companies lose 6-9 months of salary per resignation in recruitment and training.
This project predicts which employees are likely to leave before they resign.

## Key Findings
- **16.1%** overall attrition rate (237 out of 1,470 employees)
- Employees doing overtime resign at **3x the rate** (30.5% vs 10.4%)
- Monthly income gap: **$2,046** between resigned and active employees
- ML model accuracy: **ROC-AUC = 0.80**
